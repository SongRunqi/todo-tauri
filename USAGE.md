# 🚀 运行指南

## 快速开始

### 方式一：使用启动脚本（推荐）

```bash
./run-dev.sh
```

脚本会自动检查环境并启动应用。

### 方式二：手动运行

```bash
npm run tauri dev
```

## ⚙️ 首次使用配置

应用启动后：

1. **点击右上角的 ⚙️ 设置按钮**
2. **输入你的 DeepSeek API 密钥**
   - 获取地址：https://platform.deepseek.com/
   - 注册账号后在 API Keys 页面创建
3. **选择语言**（默认中文）
4. **点击"保存设置"**

## 📝 使用示例

配置完成后，你可以用自然语言创建任务：

### 简单任务
```
"买菜"
"写报告"
```

### 带截止日期的任务
```
"明天下午之前完成项目报告"
"下周五给客户打电话"
```

### 循环任务
```
"每周一早上9点团队会议"
"每天晚上8点运动"
"每月1号交房租"
```

AI 会自动解析：
- ✅ 任务名称
- ✅ 截止日期
- ✅ 优先级
- ✅ 循环模式

## 🎯 功能说明

### 任务管理
- **添加任务**：在输入框输入内容，按回车或点击"添加"
- **完成任务**：点击任务前的复选框
- **删除任务**：点击任务右侧的"删除"按钮

### 过滤功能
- **全部**：显示所有任务
- **进行中**：只显示未完成的任务
- **已完成**：只显示已完成的任务

### 底部功能
- **任务统计**：显示未完成任务数量
- **清除已完成**：一键删除所有已完成的任务

## 🔧 故障排除

### 问题 1：应用无法启动

**解决方案**：
```bash
# 清理并重新构建
cd src-tauri
cargo clean
cd ..
npm run tauri dev
```

### 问题 2：无法创建任务

**原因**：未设置 API 密钥

**解决方案**：
1. 点击右上角设置按钮
2. 输入 DeepSeek API 密钥
3. 保存设置后重试

### 问题 3：端口占用

**解决方案**：
```bash
# 查找占用端口的进程
lsof -i :1420

# 结束进程
kill -9 <PID>

# 重新启动
npm run tauri dev
```

## 📦 构建生产版本

开发完成后，可以构建独立的可执行文件：

```bash
npm run tauri build
```

构建完成后，可执行文件位于：
- Linux: `src-tauri/target/release/todo-tauri`
- macOS: `src-tauri/target/release/bundle/macos/TodoApp.app`
- Windows: `src-tauri/target/release/todo-tauri.exe`

## 🌐 切换 AI 提供商（可选）

如果想使用 OpenAI 或其他 LLM：

1. 点击设置按钮
2. 填写以下信息：
   - **API 密钥**：你的 OpenAI API 密钥
   - **LLM API 端点**：`https://api.openai.com/v1/chat/completions`
   - **模型**：`gpt-4` 或 `gpt-3.5-turbo`
3. 保存设置

## 📍 数据存储位置

- **配置文件**：`~/.local/share/com.todoapp.dev/config.json` (Linux)
- **任务数据**：`~/.todo/todos.json`
- **已完成任务**：`~/.todo/backup.json`

## 💡 小贴士

1. **自然语言输入**：尽量用自然的方式描述任务，AI 会理解
2. **截止日期识别**：支持"明天"、"下周五"、"本月底"等表达
3. **循环任务**：说明"每天"、"每周"等关键词，AI 会自动创建循环任务
4. **快捷键**：输入框中按 Enter 快速添加任务

3. 确认网络连接正常
